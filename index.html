work experience

i'm currently interning at a Thai startup called Guestasy, and i've been there for like, 7 months now. i guess you could call me a software engineer, idk. i've mostly been working on this one big project: their report generator.

basically, we help hotels manage their online reputation. we do stuff like:

  • responding to reviews: they respond to reviews left by customers on their client's page on review platforms in various languages
  • monthly reports: they send out these reports every month to their clients, detailing their performance, things like how many reviews they got, what languages people are writing in, etc.

the old way of making these reports was very tedious. you had to manually query all the data from MongoDB, feed the reviews to some AI to analyze them, and copy and paste everything into canva. it was a pain to do this for 50+ hotels.

so, i built this automation system using Python to do all of that automatically. the code is shit, but here's how it works:

  • query data: it goes to MongoDB and quries all the review data it needs like the review count, rating, etc.
  • ai magic: the reviews are then passed onto the OpenAI api to do some stuff like:
    • sentiment analysis: gets key words from the review and their sentiment (positive, negative, neutral).
    • staff mentions: detects the mentions of any staff in reviews and analyzes their performance
    • word clouds: generates a word cloud of positive and negative phrases
    • summaries: it writes summaries about different things, like what people are saying about the staff, what needs improvement, what's good, etc.
  • generation time: it takes all the data and the AI insights, and automatically creates a PowerPoint presentation using this library called python-pptx. then, it turns it into a PDF and uploads it to S3 so we can send it to the clients.

it's a fun project to be working on, and i've learned a lot about Python libraries like openai and pymongo. i'm always adding new features, fixing my shit code, and making it faster. the whole thing is over 4000 lines of code now, and it's still growing.

personal projects

my main project so far is rankPredictor. it's a website which allows you to predict your rank for various entrance exams by inputting your marks. it currently has data for MET 2025 & 2024, KCET 2024. i had made it with a friend while anxiously waiting for my MET result. it works on the basis of linear interpolation. data is scraped from the web and and college campuses for various places, and the user's score is interpolated based on the scraped data to return a rank. it utilises simple HTML, CSS and Javascript with no bloatware frameworks like react. for MET 2024, it was very accurate for ranks greater than 7k with only a 1k difference in predicted and actual rank. you can check it out at https://rankpredictor.in
